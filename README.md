## 联通营业厅青岛啤酒自动激活程序

### 需要python3
#### 流量激活需要两分钟之后再激活

#### 使用方法 

##### 添加自己的appid和deviceid 安装好python3，之后输入
```bash
[root@server]# ./CnUnicom.sh 电话号 密码
```
